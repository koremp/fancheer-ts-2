* 2022.01 (1인 개인 프로젝트)

* TypeScript, React.js, Next.js, Ant-Design, GitHub Actions, gh-pages, Jest, etc

## Setup & Usage

```
# git clone
git clone https://github.com/koremp/fancheer-ts-2

# install JavaScript dependencies
npm install

# run dev server
npm run dev
```
